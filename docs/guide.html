アスリート情報技研

　　スポーツ選手検索｜使い方ガイド

　　選手名だけでも検索できます。絞り込み機能を使うと、より探しやすくなります。

基本的な使い方

  1. 選手名を入力（※名前だけでも検索できます）
  2. 競技や性別で絞り込み検索（任意）
  3. 検索ボタンを押す
  4. 条件に合う選手が表示されます

①検索のポイント

選手名はフルネームでなくても検索可能です。
名前だけで検索し、結果が多い場合は絞り込み機能を活用すると便利です。

②絞り込み機能について

  • 競技（例：サッカー選手、野球選手など）
  • 性別（男性・女性）

複数条件を組み合わせることで、検索結果が効率よく絞り込めます。

③注意点

  • データが存在しない項目は「不明」と表示される場合があります
  • 完全一致ではなく、部分一致で検索されます

④検索例（例：大谷翔平）

  • フルネーム検索：「大谷翔平」
  • 部分一致検索：「大谷」、「翔平」
  • 部分一致＋競技名：「大谷」＋「野球選手」
  • 競技名＋性別：「野球選手」＋「男性」

⑤ お気に入り機能

各選手カードに表示されている ★（星）アイコン を押すことで、 選手をお気に入りとして登録できます。

  • ★が表示されている状態：お気に入り登録済み
  • 再度クリックすると解除されます

お気に入り情報はブラウザに保存され、ページを再読み込みしても保持されます。

⑥ 競技ジャンル検索

競技ジャンルページでは、競技ごとのボタンが一覧で表示されます。

ボタンを押すと、その競技に該当する選手のみが検索結果として表示され、 競技名を入力しなくても簡単に選手を探すことができます。

⑦ ランキング機能

ランキングページでは、登録されている選手データをもとに、 条件別のランキングを確認できます。

  • 身長が高い選手
  • 体重が重い選手

人気の高い選手を一覧で把握することができます。

⑧ 統計・データ表示

統計ページでは、選手データを集計し、競技別や性別ごとの人数などを 分かりやすく確認できます。

  • 競技ごとの選手人数
  • 男女比の傾向
  • 全体データの特徴
